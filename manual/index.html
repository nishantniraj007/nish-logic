3 OVER MATCH FREE GitHub Pages

// Games Vault — Competitive Exam Drill

3 Over Match is Nish-Logic's flagship exam drill engine. Built for serious aspirants preparing for SSC, Bank, UPSC, and CAT, it fires 18 live questions pulled directly from the Nish-Logic Firestore vault — a mix of Quantitative Aptitude, Logical Reasoning, Static GK, and Current Affairs — all calibrated to your chosen difficulty level.

Each match is 3 overs of 6 questions — hence the name. You get a randomised set every time, a 15-minute optional timer to simulate exam pressure, Hindi translation for regional accessibility, and a Certificate of Performance at the end with full answer explanations and memory tricks.

No login. No install. No cost. Just open and play.

AI NOVEL WRITER BYOK Firebase Hosting

// Tools — Bring Your Own Key

The Nish-Logic AI Novel Writer is a browser-based creative writing tool powered by Google Gemini. It runs entirely in your browser — no server, no subscription. You bring your own Gemini API key and the tool handles the rest. Write chapters, generate story continuations, and build full narratives without paying for any SaaS.

// HOW TO GET YOUR FREE GEMINI API KEY

  1. Open aistudio.google.com in your browser
  2. Sign in with your Google account
  3. Click Get API Key in the left sidebar
  4. Click Create API Key
  5. Select Create API key in new project
  6. Your API key will be generated — copy it immediately
  7. Paste it into the API Key field in the AI Novel Writer tool
  8. Your key is stored only in your browser — never sent to any server

Note: Google AI Studio free tier gives you generous daily limits. No billing required for personal use.
