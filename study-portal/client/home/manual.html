🔑 How to Get Your Free AI API Key

Some tools like AI Novel Writer require a free Google AI Studio API key. Here's how to get one in under 2 minutes:

1
Open your browser and go to aistudio.google.com
2
Sign in with your Google account
3
Click Get API Key in the left sidebar
4
Click Create API Key → select any project → copy the key
5
Paste the key into the tool when prompted — it stays on your device only
⚡ Note: The free tier gives generous daily usage — more than enough for personal use. Your key is never stored on any Nish-Logic server.
