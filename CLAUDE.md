# Nish-Logic Global Context

## Project Overview
A multi-platform hub for AI-powered tools and cricket-themed games.

## 🏛️ Permanent Architectural Rules
- **Main Portal**: Hosted at [nish-logic.web.app](https://nish-logic.web.app)
- **Firebase Hosting**: All **Tools** (e.g., AI Novel Writer) must be served from Firebase.
- **GitHub Hosting**: All **Games** (e.g., 3 Over Match) must be served from GitHub Pages at `nishantniraj007.github.io/nish-logic/`.
- **Content Pipeline**: Daily quiz questions are generated by Gemini via GitHub Actions and saved as static JSON.

## Context Routing
→ games: games/CLAUDE.md
→ tools: tools/CLAUDE.md
→ backend: server/CLAUDE.md
